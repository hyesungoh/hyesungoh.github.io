#### 기본 문법
{% highlight python %}
   # var = value if condition else value
   num = 3 if 5 > 3 else 5
   # num = 3
{% endhighlight %}


condition의 값이 True일 시 if 앞, False일 시 else 뒤


#### 사용
{% highlight python %}
   print("Hello") if True else print("World")
   # prints "Hello"
{% endhighlight %}
